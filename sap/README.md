# SAP資格クイズ

SAP S/4HANA / SAP Technology Associate / SAP Development Associate / SAP Integration Associate 対応のクイズサイトです。

## 仕様

- 4カテゴリ対応
- 表示は50問ランダム
- 問題文重複を除外して出題
- `window.quizData.s4hana` のようにJSから読み込み
- 回答後、自動で次の問題へ移動
- 左右にAdSense広告枠あり
- PC / スマホ対応

## URL例

```text
index.html
index.html?type=s4hana
index.html?type=technology
index.html?type=development
index.html?type=integration
```
